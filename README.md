## 📖 项目介绍

SystemDo 软件开发指南 — 涵盖全栈工程、软件架构模式和 DevOps 实践的完整知识体系。汇聚了我们 10+ 年服务全球海外华人企业的技术沉淀。

> **核心理念：** 技术服务于业务，而非反过来。写能维护、可扩展、与商业目标对齐的代码。

## 📂 指南目录

### 🏗️ 架构与设计

- 单体架构 vs 微服务 — 如何选择适合的架构
- 领域驱动设计（DDD）实战
- 事件驱动架构模式
- API 设计最佳实践（REST、GraphQL）
- 数据库设计与性能优化

### 💻 前端开发

- Vue 3 / Nuxt 3 最佳实践
- React / Next.js 开发模式
- 状态管理策略对比
- 组件设计系统搭建
- 前端性能优化
- Web 应用 SEO / GEO 优化

### ⚙️ 后端开发

- Node.js / Express 企业级实践
- PHP / Laravel 最佳实践
- ThinkPHP 企业开发模式
- API 认证与授权方案
- 消息队列与后台任务
- 文件存储与云服务集成

### 📱 移动端开发

- Flutter 跨平台开发实战
- uni-app 微信生态开发
- React Native 最佳实践
- 原生 Android / iOS 混合开发
- 蓝牙与 IoT 设备通信

### 🤖 AI 集成开发

- OpenAI / ChatGPT API 企业集成
- Claude API 企业工作流
- RAG（检索增强生成）系统搭建
- AI Agent 架构设计
- MCP（模型上下文协议）实战
- 企业级 AI 知识库构建
